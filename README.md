# vue-cli-starter

一个开箱即用的基于 `Vue-Cli` + `Vue2` + `Vuex` + `Vant` + `TailwindCSS` + `TypeScript` 的项目模板

这个工程的搭建笔记可以[在这里查看](https://welives.github.io/blog/front-end/engineering/vue-cli.html)

## 使用

```sh
pnpm install
```

### 开发模式

```sh
pnpm run serve
```

### 打包

```sh
pnpm run build
```

### 代码规范检查

```sh
pnpm run lint
```
